# Instrucciones para Correr Backend
    1. Ingresar a la carpeta \SIGC\backend\
    2. Abrir la terminal cmd
    3. Iniciar el entorno virtual usando el comando .\.venv\Scripts\activate 
    4. Recargar el backend usando el comando uvicorn app.main:app --reload
    4. Dirigirse al puerto indicado para probar con SWAGGER
